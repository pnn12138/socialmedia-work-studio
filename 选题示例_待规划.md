## 2 设计统一的图片搜索接口

设计一个 **平台无关的图片检索接口**。

接口名称：

searchImages

输入参数：

{  
  query: string  
  orientation?: "portrait" | "landscape" | "square"  
  color?: string  
  size?: "small" | "medium" | "large"  
  per_page?: number  
  page?: number  
}

统一输出格式：

{  
  source: "pexels"  
  query: string  
  results: [  
    {  
      id: string  
      title: string  
      photographer: string  
      photographer_url: string  
      page_url: string  
      image_url: string  
      thumb_url: string  
      width: number  
      height: number  
      alt: string  
    }  
  ]  
}

注意：

- 不要暴露原始 API 结构
    
- 不同平台必须返回统一格式
