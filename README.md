# AI 图文创作工作流

> 定位：AI 工具 × 效率方法 × 研究生生活

一个面向中文图文自媒体的内容创作工作流，专为个人创作者设计，配合 Claude Code 实现高效协作。

---

## 快速开始

### 1. 新建草稿

```bash
# 复制小红书模板
cp templates/小红书模板.md 10-content/2-drafts/20250307-你的选题.md
```

### 2. 开始写作

用你喜欢的编辑器打开草稿文件，按照模板结构填充内容。

### 3. 发布归档

```bash
# 发布后移动到已发布目录
mv 10-content/2-drafts/20250307-你的选题.md 10-content/3-published/20250307-小红书 - 你的选题.md
```

## 与 Claude Code 协作

### 常用场景

| 场景 | 指令示例 |
|------|------|
| 优化标题 | "帮我优化这个小红书标题，更吸引点击" |
| 检查草稿 | "检查这篇草稿的结构和节奏" |
| 生成变体 | "把这篇内容改写成公众号长文" |
| 提取知识卡 | "从这篇文章提取 3 张知识卡" |
| 分析数据 | "帮我分析这篇爆款的原因" |

### 协作原则

1. **人主 AI 辅**：创意和观点来自你，AI 负责优化表达
2. **模板先行**：用模板保证质量下限
3. **快速迭代**：先完成再完美，发布后复盘优化
