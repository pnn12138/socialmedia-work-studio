# 📱 Obsidian 手机同步配置教程

## 方案：Git + Remotely Save 插件

---

## 第一步：在手机上安装插件

### Android / iOS 通用步骤：

1. 打开 Obsidian 设置
2. 进入 **第三方插件** → 关闭 **安全模式**
3. 点击 **浏览** → 搜索 `Remotely Save`
4. 点击 **安装** → 启用插件

---

## 第二步：配置 GitHub 同步

### 2.1 获取 GitHub Token

1. 访问 https://github.com/settings/tokens
2. 点击 **Generate new token (classic)**
3. 填写备注：`Obsidian Sync`
4. 勾选权限：✅ **repo** (完整权限)
5. 点击 **Generate token**
6. **复制 token**（只显示一次，保存好！）

### 2.2 配置 Remotely Save

1. 打开 Obsidian 设置 → **Remotely Save**
2. 选择服务类型：**S3** 或 **WebDAV**（推荐用 **S3 兼容**）

或者使用 **Git 插件**（更简单）：

1. 安装 **Obsidian Git** 插件
2. 设置 → **Obsidian Git**
3. 填写：
   - **GitHub Repository**: `pnn12138/socialmedia-work-studio`
   - **Token**: （刚才复制的 token）
4. 点击 **Validate** 测试连接
5. 启用 **自动同步**：
   - ✅ Auto backup on interval（每 5 分钟）
   - ✅ Auto pull on startup（启动时拉取）

---

## 第三步：首次同步

### 在电脑上：
```bash
cd C:\Users\pnn\Desktop\socialmedia-work-studio
git add .
git commit -m "准备手机同步"
git push
```

### 在手机上：
1. 打开 Obsidian
2. 下拉刷新（或点击 Git 插件的 Pull）
3. 等待同步完成

---

## 第四步：验证同步

1. 在电脑上新建一个笔记
2. 在手机上刷新，应该能看到
3. 在手机上修改笔记
4. 在电脑上刷新，应该能看到

---

## ⚠️ 注意事项

### 避免冲突：
- ❌ 不要同时在手机和电脑编辑同一个文件
- ✅ 编辑前确保已同步最新内容
- ✅ 每天至少手动同步一次

### 排除文件：
已配置 `.gitignore` 排除以下内容（不会同步）：
- `.obsidian/workspace.json`（窗口布局）
- `.obsidian/hotkeys.json`（快捷键）
- `.venv/`（Python 虚拟环境）
- `__pycache__/`（缓存文件）

---

## 🔧 故障排除

### 同步失败：
1. 检查网络连接
2. 检查 GitHub token 是否过期
3. 手动执行 `git pull` 看错误信息

### 文件冲突：
1. 保留最新版本
2. 删除冲突文件（`.git/` 开头的）
3. 重新同步
